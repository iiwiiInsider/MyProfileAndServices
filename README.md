## Trading Journal

A professional Excel-based trading journal designed for intraday traders to track, analyze, and improve trading performance.

**Features:**
- 📊 Comprehensive trade logging with automatic P&L calculations
- 📈 Weekly performance dashboard with key metrics
- 📉 Monthly summary and trend tracking
- 🎯 Performance analytics by setup, time, and symbol
- 📝 Trading setup documentation and playbook

See [TRADING_JOURNAL_GUIDE.md](TRADING_JOURNAL_GUIDE.md) for detailed usage instructions.
